# 数据仓库 (Data Warehouse)
```
用途: 为所有系统提供数据支撑,决策支撑

数据来源：业务系统数据库，日志采集系统，爬虫系统

做什么： ETL 拆分，统计，明细汇总...

输出: 报表，用户画像，风控，机器学习...
```

## 离线数仓架构体系
###  数仓分层
  * ODS (原始数据层)
  保持原始数据 ，不对数据进行处理
  * DWD (明细数据层)
  对ODS层数据进行清洗(去除空值，脏数据...)
  * DWS (服务数据层) 
  对DWD为基础，进行轻度汇总
  * ADS （应用数据层）
  对DWD层汇总 直接提供应用数据
